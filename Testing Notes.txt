What production projects expect


npm install -D jest ts-jest @types/jest
npm install -D supertest @types/supertest
npx ts-jest config:init


You should have three levels:

Unit Tests
Test services (business logic)
Mock DB / cache / queue

Examples

Todo creation rules
Status transitions
Validation logic
Permission checks

Test Data Management 

Rules

No shared DB data

-----------------------------------------------------
Negative & Edge Case Tests (this separates pros)

Add tests for:

Empty input
Duplicate todo
Unauthorized access
Rate-limited request
Invalid pagination param

CI Integration (mandatory)

Your CI should:

Install deps
Start test infra
Run unit tests
Run integration tests
Enforce coverage threshold
